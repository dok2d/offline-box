# Kanban

Минималистичная Kanban-доска для управления задачами. Go-приложение с SQLite, поддержкой спринтов, эпиков, тегов, комментариев с Markdown, уведомлений и Telegram-интеграцией. Встроенная аутентификация с ролями (Admin, User, Read-only).

## Переменные

| Переменная | Описание | Значение по умолчанию |
|-----------|----------|----------------------|
| `enable_kanban` | Включить/выключить сервис | `true` |
| `kanban_port` | Порт веб-интерфейса | `8019` |
| `kanban_data_dir` | Каталог для хранения данных | `/opt/kanban/data` |
| `kanban_config_dir` | Каталог конфигурации | `/opt/kanban/config` |
| `kanban_upload_max_size` | Максимальный размер загрузки | `10m` |
| `kanban_git_url` | URL Git-репозитория | `https://github.com/dok2d/kanban.git` |
| `kanban_git_ref` | Ветка/коммит для клонирования | `main` |

## Порт

Сервис слушает на `127.0.0.1:8019`. Доступен через nginx по пути `/kanban/`.

Nginx использует `sub_filter` для перезаписи абсолютных путей в ответах (`/api/`, `/static/`, `/images/`, `/files/` → `/kanban/...`), т.к. приложение не поддерживает sub-path нативно.

## Данные

- **Данные**: `{{ opt_base }}/kanban/data`
  - `kanban.db` — единственный персистентный файл. SQLite-база содержит все данные: задачи, пользователи, комментарии, спринты, эпики, а также загруженные файлы и изображения (хранятся как BLOB)
- **Конфигурация**: `{{ opt_base }}/kanban/config` — Containerfile, клонированный исходный код

## Инициализация

При первом развёртывании выполняются следующие действия:

1. Создание каталогов для данных и конфигурации
2. Клонирование исходного кода из Git (или копирование из offline-зависимостей)
3. Сборка контейнерного образа `localhost/kanban` (multi-stage: загрузка ассетов → сборка Go → runtime на debian:13-slim)
4. Создание и запуск systemd user unit (read-only rootfs, tmpfs для /tmp)
5. Ожидание готовности сервиса
6. Настройка nginx reverse proxy с sub_filter

При первом входе через браузер будет предложено создать учётную запись администратора.

## Зависимости

- Базовый образ: `docker.io/library/debian:13-slim` (runtime) + `golang:1.22-bookworm` (build stage)
- Go-зависимости: `mattn/go-sqlite3` (единственная)
- Роли: `base`, `podman`, `nginx`
