# SearXNG

Мета-поисковая система с фокусом на приватность. Агрегирует результаты из множества поисковых систем. В офлайн-режиме может работать с локальными источниками данных, если они настроены.

## Переменные

| Переменная | Описание | Значение по умолчанию |
|-----------|----------|----------------------|
| `enable_searxng` | Включить/выключить сервис | `true` |
| `searxng_port` | Порт веб-интерфейса | `8009` |
| `searxng_data_dir` | Каталог для хранения данных | `/opt/searxng/data` |
| `searxng_config_dir` | Каталог конфигурации | `/opt/searxng/config` |
| `searxng_secret_key` | Секретный ключ для подписи сессий | (из passwords.yml) |
| `searxng_base_url` | Базовый URL-путь | `/searxng/` |

## Порт

Сервис слушает на `127.0.0.1:8009`. Доступен через nginx по пути `/searxng/`.

## Данные

- **Данные**: `{{ opt_base }}/searxng/data` -- каталог данных
- **Конфигурация**: `{{ opt_base }}/searxng/config` -- `settings.yml`, Containerfile

## Инициализация

При первом развёртывании выполняются следующие действия:

1. Создание каталогов для данных и конфигурации
2. Генерация `settings.yml` из шаблона с заданными параметрами
3. Сборка контейнерного образа `localhost/searxng` из Containerfile
4. Создание и запуск systemd user unit
5. Настройка nginx reverse proxy

## Зависимости

- Базовый образ: `docker.io/library/debian:13-slim`
- Роли: `base`, `podman`, `nginx`
